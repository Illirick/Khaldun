## Controls

This game doesn't really have a tutorial so here is the contorol scheme:

- Movement: Arrow keys
- Jump: Spacebar or X
- Attack: Z
- Dash: Left Shift
- Change weapon: A to cycle, numbers to choose specific
- Restart: R
